# IIDX　楽曲　特徴

## 概要

音楽ゲーム「beatmania IIDX」の楽曲の取り扱いについてまとめる。


### 楽曲プレイの可否

beatmaniaIIDXでは、それぞれの環境において、プレーできる楽曲、プレーできない楽曲が存在する。プレー環境は今回は2つに分類する。

- AC：アーケード版のIIDXをプレーする環境
- INFINITAS1:KONAMIからリリースされているPC版のIIDXをプレーする環境

なお、旧バージョンにおいてはPS2でそれぞれのバージョンのソフトがリリースされているが、今回はその環境は対象外とする。

AC, INFINITASそれぞれの環境において、プレーできる楽曲、プレーできない楽曲は、以下のURLのファイルにまとまれている。

https://chinimuruhi.github.io/IIDX-Data-Table/textage/chart-info.json

これは、各楽曲IDをキーとして、以下の要素を含めたデータである。

- bpm：楽曲のBPM（Beats Per Minute）
- in_ac：ACでプレーできるかどうか（true/false）
- in_inf：INFINITASでプレーできるかどうか（true/false）

AC、またはINFINITASでのプレーができるかの可否は、in_ac、in_infそれぞれにおいて、truaであるかどうかを見ること。


### 1つの楽曲に対して複数の楽曲IDが存在するパターン

今回用いるデータには、1つの楽曲に対して複数の楽曲IDが存在するパターンがある。例えば、楽曲「New Castle Legions」には、以下の2つの楽曲IDが存在する。

- 1833
- 1877
  
この場合、楽曲IDごとに、AC、INFINITASでプレーできる譜面が異なる場合があることを示している。本楽曲の場合、楽曲IDが1833のものはINFINITASでプレーできる譜面、1877のものはACでプレーできる譜面が存在することを示している。

AC、INFINITASでプレーできる譜面が異なる楽曲がある経緯として、AC版のIIDXで削除された楽曲が、再びAC版で収録された（その楽曲を以下、復活曲と呼ぶ）ことが挙げられる。復活曲の場合、再び収録された際に譜面が変更される場合がある。（全ての難易度（NORMAL、HYPER、ANOTHER）で譜面が変更されるとは限らず、一部譜面だけが変更される場合もある）また、復活された際に、一部の楽曲に搭載されている難易度`LEGGENDARIA`の譜面が追加される場合もある。

そのような経緯から、1つの楽曲に対して複数の楽曲IDが存在するパターンがあることを理解しておく必要がある。

実装においては以下の点を留意すること

- 、1つの楽曲に対して複数の楽曲IDが存在する場合、候補は重複させず、1つにマージすること。
  - 例：New Castle Legionsの場合、ACでプレーできる譜面とINFINITASでプレーできる譜面が異なるため、データとしては2つ存在しているが、候補としては「New Castle Legions」として1つにまとめること。

- 今後の実装にあたり、ある楽曲に対してそれぞれの難易度の譜面をTexTage側で作成された譜面のURLにリンクさせたい。その場合、AC版とINFINITAS版の譜面が異なる場合は、それを区別させてリンクさせること。
