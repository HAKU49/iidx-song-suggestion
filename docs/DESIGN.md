# IIDX　楽曲検索アプリ　デザイン仕様書

## 概要
IIDX楽曲検索アプリにおける、デザイン、UI/UXの仕様の方針をまとめる。

### デザイン方針

#### カラー
色については以下の4パターンをユーザー側で指定できる形にする。以下の4パターンを画面の右上に切り替えボタンを配置して、ユーザーがいつでも切り替えられるようにする。
- ライトモード
- ダークモード
- Sparkle Showerモード（黄緑、 黄色のカラーをメインにする）
- tricoroモード（サイバー空間を模した白よりの灰色を背景に、赤、青、黄の3色のカラーを用いる）

#### トップ1件の表示について
トップ1件で表示させる曲名は、他の文字と区別がつきやすいデザインにすること

### フォント
画面上に表示させるフォントについては、それぞれ以下のフォントを利用すること。ただし、提示したフォントについては無償利用可能なものを利用すること。参考用にURLで示したフォントはライセンスの購入が必須のものがあるため、購入が必須の場合は、代替となる無償利用可能なフォントを探して、それを利用すること。ライセンス違反は絶対に行わないこと。

- アルファベット：Orbitron
- 日本語：Noto Sans JP
