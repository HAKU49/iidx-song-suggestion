このアプリについて

♫

ABOUT

beatmania IIDX の楽曲を、あいまいな入力からでも検索できるアプリです。

  • 曲名を正確に覚えていなくても、スペルミスや読み仮名でも検索できます
  • コミュニティで使われている略称・スラングからも検索できます
  • AC版（アーケード）/ INFINITAS の絞り込みフィルターに対応しています
💾

DATA SOURCE

以下のデータを使用させていただいています。

  • chinimuruhi 様 が公開されている IIDX 関連データ
    • 楽曲一覧・難易度情報・バージョン情報などの JSON データ
  • TexTage 様（管理人：泡井七月 様）のデータ
    • 上記データのクロール元として利用されているデータベースサイト
🚀

PLANNED FEATURES

  • 各楽曲に TexTage 様の譜面ページへのリンクを追加予定 WIP
  • 曲名略称の追加・検索性の向上 WIP
    • 2026/4/2 時点では一部楽曲のみ略称を登録済み
    • SNS などで使われている略称・スラングを順次追加予定
